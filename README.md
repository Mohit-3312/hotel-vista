# 🏨 Hotel Vista

A modern full-stack hotel management and booking platform built with **React.js** on the frontend and **Java Servlets** on the backend.

![GitHub](https://img.shields.io/badge/Status-Active-success)
![Frontend](https://img.shields.io/badge/Frontend-React.js-61DAFB)
![Backend](https://img.shields.io/badge/Backend-Java%20Servlets-orange)
![Database](https://img.shields.io/badge/Database-MySQL-blue)

---

## ✨ Features

### 👤 User Features

* Browse available rooms
* View room details and pricing
* Search and filter rooms
* Secure user registration & login
* Book hotel rooms online
* View booking history
* Responsive UI for desktop and mobile

### 🛠️ Admin Features

* Manage room inventory
* Add, update, and delete rooms
* View customer bookings
* Manage hotel information
* Dashboard for administration

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Mohit-3312/hotel-vista.git
cd hotel-vista
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The React application will run at:

```text
http://localhost:5173
```

### Backend Setup

1. Import the backend project into your IDE.
2. Configure Apache Tomcat.
3. Set up database credentials.
4. Deploy the project on Tomcat Server.

Backend will run at:

```text
http://localhost:8080
```
